# Linux Tooltips Java GUI
Get tips and tricks about your Linux system using a Java GUI with a button. Tips are sourced from [https://tips.tycloud.dev](https://tips.tycloud.dev). See this repo for the code running on the server: [https://github.com/Tyguy047/Tooltip-Server](https://github.com/Tyguy047/Tooltip-Server)

## Notes:
- Update the `server` variable in `Call.java` if you have a custom tip server.
- This project may not be perfect since I am using it to learn Java and GUIs in Java.
